Data domains

Each domain is a bounded context where accountability and authority over data meaning reside. All data products — including KPIs — are owned by exactly one domain. Domain types are defined in the legend below; all types follow the same governance model.

Source-aligned Own production of data "What happened?"
Consumer-aligned Own interpretation for a use case "What does it mean?"
Aggregated Own standardized cross-portfolio view "What do we decide on?"
Supporting Shared infrastructure capability Not a spoke domain
